# portfolio

This repository contains a small GitHub Pages site scaffold in the docs/ folder.

How to publish

1. Go to the repository Settings -> Pages.
2. Under "Build and deployment" select "Source" -> "Deploy from a branch".
3. Choose the default branch (e.g., main) and set the folder to `/docs`.
4. Save. GitHub Pages will publish the content from the `docs/` folder.

What's included

- docs/index.html — homepage
- docs/about.html — about page
- docs/projects.html — projects list
- docs/assets/style.css — site styles
- docs/assets/favicon.svg — favicon
