# GitHub Pages site using the Cayman theme

This branch applies the `cayman` theme to your site. To preview locally:

1. Install Ruby and Bundler.
2. From the repo root, checkout this branch: `git checkout gh-theme-cayman`.
3. From the `docs/` folder run:
   - `bundle install`
   - `bundle exec jekyll serve --source .`
4. Open http://127.0.0.1:4000/
